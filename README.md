# 🎤 Automatic Linguistic Converter

### A Real-Time Text & Voice Translation System

---

### 📋 Project Overview
This is my **B.Sc Computer Science final year project** submitted to **Ayyan Thiruvalluvar College of Arts and Science**.

**AUTOMATIC LINGUISTIC CONVERTER** is a powerful mobile application that provides instant translation for both text and speech. It helps users communicate easily across different languages in real-time.

### ✨ Key Features
- **Real-time Voice Translation:** Speak directly and get instant translation
- **Text Translation:** Quick and accurate text input translation  
- **Cross-Platform:** Works smoothly on both Android and iOS
- **Responsive UI:** Clean, interactive user interface

### 🛠️ Tech Stack
- **Frontend:** React Native
- **Backend:** FastAPI (Python)
- **Database & Search:** Elasticsearch for smart, contextual translations
